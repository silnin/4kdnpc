# 4kdnpc
NPC generator for my "4 Kingdoms" campaign

# Installation

## npm
* npm install

## database
* install mysql: sudo apt-get install mysql-server-5.5
* create a database
* rename src/db-dist.js to src/db.js
* update info in db.js

# Start
* start server: node ./app.js
* browse to localhost:3000
